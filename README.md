# bones
The bones language...

bones is designed for people ages 8 to 88 for the purpose of data wrangling and algo development. it is designed 
to help programmers help people and not as a general-purpose programing language.

This repo will be sparce for a while and may well be a waste of time unless you are a collaborator but feel free 
to enjoy any bits you find helpful...


GOALS

1) pybones - a python implementation of bones where the standard library is implemented in python, where python can
   call bones and bones call python
